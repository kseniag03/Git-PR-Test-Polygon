# Классы и объекты (ООП)

## Класс Account
1. моделирует поведение банковского счёта
2. выполняет базовые операции (пополнение, снятие, показ баланса),
3. ведёт учёт всех действий, а также предоставляет историю операций

## Класс CreditAccount(Account)
1. наследует всю функциональность класса Account
2. при инициализации принимает дополнительный параметр credit_limit: баланс такого счета может быть отрицательным, но не ниже значения -credit_limit
3. по запросу показывает, сколько кредитных средств еще доступно
4. в историю операций добавляется информацию о том, были ли использованы кредитные средства
